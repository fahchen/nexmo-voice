# Nexmo::Voice

A Ruby wrapper for the Nexmo Voice API.

## Installation

Add this line to your application's Gemfile:

    gem 'nexmo-voice'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install nexmo-voice
